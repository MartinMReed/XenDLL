XenDLL



Config File Options:

   1. "menuLocation" = Options: TOP_LEFT, TOP_RIGHT, BOTTOM_LEFT, BOTTOM_RIGHT
                       Default: BOTTOM_RIGHT
                       The Location of the on-screen menu button

   2. "menuToggle" = Options: *Most* of the DIK keys are usable (ie DIK_Q, DIK_ESCAPE, etc)
                     Default: 0x00 (nothing)
                     For a list of keys, visit: http://calc.xentales.com/HGooey/DIK.html

   3. "Computron" = Options: true, false
                    Default: false
                    Load Computron (xenimus bot). Requires registration and logging into the Computron system.

   4. "Screenshotr" = Options: true, false
                      Default: true
                      Load the screenshot saver.

   4. "AutoClick" = Options: true, false
                    Default: true
                    Load the auto-click extension.



Description / Notes:

   1. clean.bat will clear out all config xml files (except Memory.xml) for XenDLL and built-in extensions.
