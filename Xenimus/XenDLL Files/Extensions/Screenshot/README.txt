Screenshot Saver (Screenshot) [Built In]



Memory Addresses Used:

      none



Config File Options:

   1. "filePath" = Default: screenshots (in your xenimus folder)
                   The location to save the screenshots.

   2. "screenshotQuality" = Options: 1 - 100
                            Default: 100
                            The quality (percent) of the image taken.

   2. "includeGUI" = Options: true, false
                     Default: true
                     Include the XenDLL GUI objects in the screenshot.



Description / Notes:

   1. Screenshots will be saved when using the print screen key.
