Auto Click (AutoClick) [Built In]



Memory Addresses Used:

      none



Config File Options:

   1. "MenuItemIO" = Options: true, false
                     Default: false
                     Whether the extension is active or not. This will be set automatically everytime the game exits

   2. "trigger" = Options: MButton, LButton, RButton, or *most* of the DIK keys (ie DIK_Q, DIK_ESCAPE, etc)
                  Default: MButton
                  For a list of keys, view: XenDLL Files/DIK.html



Description / Notes:

   1. Setting a key for AutoClick will send a left mouse click, and not repeat the key stroke.
