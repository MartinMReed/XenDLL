XenDLL Download & Install

1. Save the XenDLL zip file


2. Open the extracted XenDLL folder
3. Copy the three files from the XenDLL folder to your Xenimus folder

Extension Download & Install

1. Save the Extension zip file


2. Open the HGooey Extension folder located in "HGooey Files\XenDLL" in your Xenimus folder


3. Open the downloaded, and extracted Extension folder
4. Copy the extension folder containing the "HG_SDK.dll" and any images (not the source) into the HGooey Extension folder

Memory File Update Download & Install

1. Find your way to the thread containing the newest memory addresses


2. Download the Memory xml file linked at the beginning of the thread


3. Open the HGooey Common folder located in "HGooey Files\XenDLL" in your Xenimus folder


4. Copy the Memory file you just downloaded into the HGooey Common folder
5. Replace the old Memory file

Load Extension InGame

1. Open the menu


2. Click the icon that looks like this


3. Click the "LOAD" button


4. Select the extension to load
5. Click "OK"
